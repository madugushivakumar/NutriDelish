# NutriDelish Frontend

React frontend for NutriDelish AI Food Platform built with Vite and TypeScript.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file (optional):
```
VITE_API_URL=http://localhost:5000/api
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Features

- Restaurant browsing and search
- Dish details and filtering
- Shopping cart
- Order placement
- Payment processing
- AI food recommendations
- Group ordering
- Wallet management
- Order tracking

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Axios (for API calls)
- Recharts (for charts)
- Lucide React (for icons)
